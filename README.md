## 🗂️ 툴 목록

### 📝 텍스트 (5개)
| 툴 | 설명 |
|----|------|
| 글자 수 세기 | 글자, 단어, 줄 수 즉시 확인 |
| 텍스트 비교 | 두 텍스트의 차이를 시각적으로 비교 |
| 대소문자 변환 | 대문자, 소문자, camelCase, snake_case 변환 |
| 줄 정렬/중복제거 | 텍스트 줄 정렬 및 중복 제거 |
| 더미 텍스트 생성 | Lorem Ipsum / 한국어 더미 텍스트 |

### 🖼️ 이미지 (5개)
| 툴 | 설명 |
|----|------|
| 이미지 리사이즈 | 원하는 크기로 조절 |
| 포맷 변환 | PNG ↔ JPG ↔ WEBP |
| 이미지 압축 | 용량 줄이기 |
| 이미지 자르기 | 원하는 영역만 크롭 |
| 이미지 → Base64 | Base64 문자열로 변환 |

### 📄 PDF (2개)
| 툴 | 설명 |
|----|------|
| 이미지 → PDF | 이미지 여러 장을 PDF로 합치기 |
| PDF → 이미지 | PDF 페이지를 이미지로 추출 |

### 👨‍💻 개발자 (5개)
| 툴 | 설명 |
|----|------|
| JSON 포맷터 | JSON 정렬, 검증, 압축 |
| Base64 변환 | 텍스트 인코딩/디코딩 |
| URL 인코딩 | URL 인코딩/디코딩 |
| 색상 코드 변환 | HEX ↔ RGB ↔ HSL |
| 해시 생성기 | SHA-256, SHA-1, SHA-512 |

### 📐 계산기 (3개)
| 툴 | 설명 |
|----|------|
| 단위 변환기 | 길이, 무게, 온도, 넓이 |
| 날짜 계산기 | D-day, 날짜 차이, 날짜 더하기 |
| 할인/세금 계산기 | 할인율, 부가세 계산 |

## 🚀 로컬 실행

```bash
# Python
python -m http.server 8080

# Node.js
npx serve .
```

브라우저에서 `http://localhost:8080` 접속
