# 🛠️ Free Tools Hub

> 설치 없이 웹에서 바로 사용하는 무료 툴 모음 사이트

모든 툴은 **브라우저에서만 실행**되며, 파일이 서버로 업로드되지 않습니다.

## 기술 스택

- HTML + Vanilla JS + CSS (외부 의존성 최소화)
- 깔끔한 화이트 미니멀 디자인
- 모바일 반응형

## 파일 구조

```
free-tools-hub/
├── index.html          ← 메인 허브 페이지 (툴 목록 + 검색)
├── style.css           ← 공통 디자인 시스템
├── tools/
│   ├── text/           ← 텍스트 도구
│   ├── image/          ← 이미지 도구
│   ├── pdf/            ← PDF 도구
│   ├── dev/            ← 개발자 도구
│   └── calc/           ← 계산기 도구
└── js/
    └── common.js       ← 공통 유틸 (클립보드, 토스트, 드롭존 등)
```

## 로컬 실행

```bash
# 방법 1: Node.js
npx serve .

# 방법 2: Python
python -m http.server 8080
```

브라우저에서 `http://localhost:8080` 접속

## 새 툴 추가하기

1. `tools/<카테고리>/` 폴더에 HTML 파일 생성
2. `../../style.css`를 import하여 공통 스타일 적용
3. `../../js/common.js`를 import하여 공통 유틸 사용
4. 공통 페이지 구조 준수:
   - 헤더: "← Free Tools" 링크 + 툴 이름
   - 메인 컨텐츠 영역
   - 모바일 반응형
5. `index.html`의 `tools` 배열에 새 툴 정보 추가:
   ```js
   { id: 'my-tool', category: 'dev', icon: '🔧', name: '툴 이름', desc: '한 줄 설명', url: 'tools/dev/my-tool.html' }
   ```

## 배포
